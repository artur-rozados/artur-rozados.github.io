Projeto: Pi-NAS

NAS baseado em Raspberry Pi com OpenMediaVault, Docker e Syncthing.
Redundância com 2 SSDs (armazenamento + backups via rsync).
Acesso remoto seguro via Tailscale.

Descrição

Desenvolvimento de um sistema de backup e sincronização descentralizado utilizando Raspberry Pi, OpenMediaVault (OMV) e Docker. O Syncthing sincroniza arquivos entre dispositivos automaticamente, enquanto dois SSDs externos garantem redundância: um para armazenamento e outro para backups incrementais via rsync.
